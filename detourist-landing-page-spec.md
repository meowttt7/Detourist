# Detourist Landing Page Spec

## Purpose
This document is the build-ready source of truth for the first Detourist landing page.

Detourist is a premium travel deal product for flexible travelers who care more about value than convenience. The landing page should make that positioning obvious within the first screen.

## Primary Goal
- Capture waitlist or early-access signups

## Secondary Goals
- Explain the product clearly in under 60 seconds
- Qualify the right audience
- Introduce the `Worth-It Score`
- Create enough trust that users want alerts

## Audience
- Travelers with moderate to high spending power
- People who want business class, first class, or luxury stays without paying full premium prices
- Users who are comfortable with repositioning, odd routings, long layovers, or departure flexibility
- Global audience, with strong relevance in Asia, Europe, Australia, and the Middle East

## Core Message
Detourist helps flexible travelers find premium flights and luxury stays that become irrationally good value if they are willing to take the smart detour.

## Tone
- Premium but not snobbish
- Smart but not technical
- Aspirational but grounded
- Clear, specific, and a little playful

## Core Terms
Use these consistently:
- `Detourist`
- `Worth-It Score`
- `Detour tolerance`
- `Why it's worth it`
- `The catch`
- `Flexible luxury`
- `Premium deal alerts`
- `Smart detour`

Avoid these as primary framing:
- `cheap luxury`
- `travel hacks`
- `budget travel`
- `mistake fares`

## CTA Strategy
Primary CTA label:
- `Get Early Access`

Secondary CTA label:
- `See Sample Deals`

Navigation CTA label:
- `Join Waitlist`

Form success message:
- `You're on the list. We'll send early access when Detourist opens up.`

## Visual Direction

### Visual Theme
Detourist should feel like:
- a premium travel editorial brand
- crossed with a smart software product

### Design Principles
- Use space generously
- Keep copy easy to scan
- Lean into travel motifs without becoming cheesy
- Make premium feel warm and human, not dark and aggressive

### Color Direction
Suggested base palette:
- Background: `#F6F0E8`
- Main text: `#1D2A2A`
- Accent teal: `#0F766E`
- Accent gold: `#B7791F`
- Soft card: `#EFE6DA`
- Border: `#D8CCBC`

### Typography Direction
- Headlines: high-character serif
- Body/UI: clean sans-serif

Example pairing idea:
- Headline: `Cormorant Garamond` or `Fraunces`
- Body: `Manrope` or `Plus Jakarta Sans`

### Imagery Direction
- route maps
- boarding pass style motifs
- premium cabin crops
- airport code snippets
- elegant editorial travel compositions

Avoid:
- generic beach stock photos
- crypto-fintech visuals
- neon luxury aesthetics

## Conversion Notes
- Put an email capture in the hero.
- Repeat capture at the end of the page.
- Secondary CTA should scroll to sample deals or open a modal with examples.
- Keep all primary CTAs consistent: `Get Early Access`

## Mobile Notes
- Hero copy should fit in 5 to 7 short lines
- Sample deals should stack vertically
- Comparison and score sections should avoid wide tables
- Accordion FAQ is preferred over long static answers

## Implementation Notes
- If the page is built in Next.js, this spec maps cleanly to a single `page.tsx` plus a `components/landing` folder.
- Keep all section content in a simple content object so copy can be changed without rewriting component markup.
- Start with static sample deals and swap in CMS-backed content later.
